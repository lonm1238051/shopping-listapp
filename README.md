# Shopping List App

간단한 쇼핑 리스트 웹 애플리케이션입니다.

## 기능

- 쇼핑 아이템 추가/삭제
- 아이템 체크(완료) 표시
- 로컬 스토리지에 데이터 저장
- 반응형 디자인 (모바일 지원)

## 사용 방법

1. `index.html` 파일을 브라우저에서 열기
2. 입력창에 아이템 이름 입력 후 "추가" 버튼 클릭 또는 Enter 키
3. 아이템 왼쪽 체크박스를 클릭하여 완료 표시
4. × 버튼을 클릭하여 아이템 삭제

## 기술 스택

- HTML5
- CSS3
- Vanilla JavaScript
- LocalStorage API
